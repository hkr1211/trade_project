# 用户指南

## 1. 买家（国际客户）

### 1.1 入门指南
1. **注册**：访问首页，点击"买家注册"。填写您的公司详情和联系信息。
2. **审批**：您的账户需要蕴杰工作人员手动审批。审批通过后，您将收到邮件通知。
3. **登录**：使用您的邮箱和密码登录。

### 1.2 管理询价
- **创建询价**：在控制面板中点击"创建询价"。
    - 填写交付要求。
    - 添加多个项目（产品名称、材料、数量）。
    - 如需要可上传图纸或附件。
- **查看报价**：蕴杰对您的询价报价后，状态会变为"已报价"。点击询价单号查看价格和交货期详情。

### 1.3 管理订单
- **下订单**：
    - **从询价创建**：进入"创建订单"，从下拉列表中选择已报价的询价。数据将自动填充。
    - **直接下单**：手动填写订单表单。
- **跟踪状态**：在订单列表中监控您的订单状态（已确认、已发货等）。
- **沟通交流**：使用任何订单或询价页面底部的"消息"部分与销售团队沟通并共享文件。

## 2. 供应商（蕴杰员工）

### 2.1 访问与角色
- **登录**：使用您的员工邮箱。
- **控制面板**：提供待处理询价和订单的高层视图。

### 2.2 处理询价
1. **接单**：进入"询价列表"。点击任何 `待处理` 的询价。
2. **报价**：
    - 输入 `交货期` 和 `供应商备注`。
    - 为每个项目输入 `单价`。
    - 点击 **提交报价**。
    - *注意*：您现在是此询价的负责人。其他销售人员无法修改它。

### 2.3 处理订单
1. **确认**：
    - 审阅 `待处理` 订单。
    - 输入 `预计交货日期`。
    - 点击 **确认订单**。
2. **更新进度**：
    - 使用操作按钮更新状态：`发货`、`确认付款`、`完成`。
    - **付款**：您必须先将订单标记为 `已付款`，然后才能将其标记为 `已完成`。

### 2.4 消息沟通
- 始终检查详情页面中的"消息"部分，查看客户问题或新上传的文件。
